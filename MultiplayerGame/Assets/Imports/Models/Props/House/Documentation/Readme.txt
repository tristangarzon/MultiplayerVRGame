Abandoned_warden_house
v1.1

Thank you for supportting this asset!

This asset contain 7 prefabs and a scene example.

"abandoned_warden_house_modular" contain 270 sub prefabs.
"abandoned_warden_house_LOD" contain a single mesh LOD (2 levels).
"abandoned_warden_house_mat_group"contain all the materials regrouped for each meshes.
"abandoned_warden_house_single_mesh" all materials regrouped in one mesh.


[Exterior version] 

"abandoned_warden_house_closed" default material grouped in one mesh
"abandoned_warden_house_closed_LOD" contain a single mesh LOD (2 levels).
"abandoned_warden_house_closed_mat_group" contain all the materials regrouped for each meshes.



I added my own colliders for the stairs/floors and wallS that you can remove it if you want.
The textures are seamless and PBR.

You can easly remove or change the materials.

I used "Sky5X One" asset for the skybox
